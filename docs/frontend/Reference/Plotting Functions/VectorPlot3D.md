Generates a 3D vector plot of the vector field `{vx,vy,vz}` as a function of $x$, $y$, and $z$
```mathematica
VectorPlot3D[{x, y, z}, {x, -1, 1}, {y, -1, 1}, {z, -1, 1}]
```
