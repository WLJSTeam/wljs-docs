Underlines text or expression

```mathematica
Squiggled[expr_, color_:Lighter[Red]]
```

It resolves into a subset of [Style](frontend/Reference/Graphics/Style.md)

Example

```mathematica
Style[If[!DictionaryWordQ[#], Squiggled[#], #], 14] & /@ 
 TextWords[ExampleData[{"Text", "TheRaven"}]][[;; 45]]
```
