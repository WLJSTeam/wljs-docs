JoyCon presenter tool

Control your presentations using JoyCon (R)

Interface​

Pair your JoyCon with your PC (unpair from Switch console firsts), and then click to connect on the single button

Then you regular interface for bindings EventHandler

EventHandler["JoyCon", {
  "zr" -> (FrontSubmit[FrontSlidesSelected["navigateNext", 1]]&),
  "y" -> (FrontSubmit[FrontSlidesSelected["navigatePrev", 1]]&),
  "Stick" -> Print (* for example *)
}];

How to enable IMU​

To read sensor data, firstly enable it using

JoyConIMU[True] // FrontSubmit

then you can read them out (both accelerometer and gyro)

EventHandler["JoyCon", {
  "IMU" -> Function[object,
	  dump = object;
  ]
}];
